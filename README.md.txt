# E-Commerce Website

This is a fully functional Ecommerce Website which allows the user to login,add products to cart,view products in detail.

Technologies: Node.js, Express, MongoDB, ejs, Bootstrap, passport.js, helmet

1.Created front-end template of the website with EJS along with HTML,CSS and Bootstrap.

2.Implemented the MVC architecture for Back-End using technologies like Node.js, express.js, MongoDB, passport.js.

3.Integrated the PayUMoney Payment gateway to process all the payments and create the orders.

4.Implemented basic Web Security features using Helmet, Mongo Sanitize etc to mitigate small attacks
